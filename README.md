# AI-pet-prototype

面向宠物电商商家的 AIGC 商品图生成工具原型。商家上传商品图，选择宠物模特和拍摄场景，AI 自动生成电商主图。纯 HTML / CSS / JS，Mock 数据，可直接在浏览器打开演示。

## 包含页面

| 文件 | 说明 |
|---|---|
| `index.html` | 首页：顶部栏 + 左侧导航 + Hero（3 图片卡片 + 3 视频按钮）+ 爆款图片/视频/模特/场景 4 行横滚 |
| `generate.html` | 图片生成工作台：顶部栏 + 左侧配置面板（上传/宠物场景Tab/提示词/吸底生成按钮）+ 右侧结果区（空态/Loading/2 图对比） |
| `styles.css` | 全站样式（TDesign 风格，主色 `#0052d9`） |
| `data.js` | Mock 数据：爆款图片/视频、10 个宠物模特、8 个场景、结果图池 |
| `home.js` | 首页交互逻辑 |
| `generate.js` | 工作台交互逻辑（上传/选择/生成/预览/登录弹窗） |
